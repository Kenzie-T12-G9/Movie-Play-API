## Movie Play API

![Logo do Projeto](./src/assets/api-logo.png)

O objetivo da API é fornecer a possibilidade de desenvolver um aplicativo de streaming com uma comunidade ativa e várias opções de customização, tanto relacionadas a moldagem do perfil de usuário, quanto com relação a adição de títulos no catálogo, avaliações e histórico de atividades.

Para mais detalhes sobre rotas e endpoints disponíveis, além de demais informações, acesse nossa [DOCUMENTAÇÃO](https://movie-play.herokuapp.com/documentation)
